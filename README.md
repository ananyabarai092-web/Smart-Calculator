# Smart Calculator CLI

Professional terminal-based Smart Calculator built with Python.

Features:
- Basic arithmetic: addition, subtraction, multiplication, division
- Advanced: power, modulus, square root, percentage, factorial, average
- Utility calculators: BMI, EMI
- Scientific functions: sin, cos, tan, log, ln, exp, sqrt
- Memory operations: M+, M-, MR, MC
- Random number generator
- History persisted to `history.json` with search/delete/clear
- User profiles persisted to `users.json` with badges
- Dark/Light theme support (Colorama)

Requirements
- Python 3.10+
- colorama

Quick start
```bash
python -m venv .venv
.venv\Scripts\activate    # Windows
pip install -r requirements.txt
python main.py
```
